Globale Produkte im Supermarkt – Website (Stand v3)
==================================================

Inhalt:
- index.html: Übersicht mit 21 Produkten
- page.html?id=fXX: Detailseiten (je Produkt)
- Editor direkt auf den Detailseiten (✎ unten rechts, Passwort 55)
  - Bearbeitet Titel, Unterzeile (Herkunft + Kurzinfo), 4 Abschnitte + Extra

Zusätzliche Seiten:
- about.html: Projektbeschreibung
- sources.html: Quellen-Vorlage

Hinweis:
Der Editor ist Frontend-only und speichert lokal im Browser.

Start:
1) ZIP entpacken
2) index.html öffnen
3) Produkt anklicken -> Detailseite
4) ✎ -> Passwort 55 -> bearbeiten -> speichern
